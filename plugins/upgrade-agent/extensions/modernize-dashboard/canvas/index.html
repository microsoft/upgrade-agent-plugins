      Assessment terms

      Issue
      A distinct rule that fired at least once. One issue can have many incidents.
      Incident
      A single location in code that triggered a rule. The total incident count is what you usually care about when sizing effort.
      Mandatory
      An issue that must be addressed for the upgrade to succeed. Other severities are advisory.
      Feature
      A higher-level capability (WCF, Remoting, etc.) that the assessment detected — usually a group of related rules.
